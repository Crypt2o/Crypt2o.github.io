About Crypt2o

Cryp2o is an advanced 2D physics extension for SFML. It offers a modular, efficient system for physics-based interactions, including collision detection, gravity, and water physics. Perfect for game development and interactive simulations.

Features

  • Gravity System: Configurable gravity for realistic movement and interactions.
  • Velocity Management: Includes max fall speed and acceleration controls for fine-tuned motion.
  • Lightweight Integration: Designed for seamless use with SFML projects.
  • Modular Design: Add only the physics components you need to keep your project efficient.
